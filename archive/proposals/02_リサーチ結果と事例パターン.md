# リサーチ結果と事例パターン(帰納分析)

2026-07-04実施。サブエージェント2体による並列リサーチ(①AI活用1人事業の事例19件、②2026年以降の日本市場トレンド)と、そこからの帰納的パターン抽出。

## 1. 事例調査の要点(19件)

### 誠実に報告すべき発見
- 「週数時間×匿名×収益が第三者検証可能」な日本の成功事例は**非常に少ない**
- note上の「月10万円達成」系は自己申告ばかりで裏付けが弱い(信頼度: 低)
- 裏付けの取れる海外成功者(Photo AI 年$3.5M、Justin Welsh 累計$10M超など)は**全員フルコミット型**
- → **過度な収益期待は禁物。**「収益より成長・おもろさ」という本事業の位置づけはデータからも正しい

## 2. 帰納で抽出した「1人AI事業の3つの型」

### 型①: 書いたものを売る(Kindle本・note有料記事)
- AIが下書き、本人が体験・データを注入。プラットフォームの検索・ランキングが集客を代行
- **フォロワーゼロでも今日から始められる**。初速は月数千円が現実線
- 失敗条件: 独自性ゼロのAI量産コンテンツは埋没(複数事例が「AI丸投げでは稼げない」と自己反省)

### 型②: 1対1で相談に乗る(壁打ち・レビュー)
- 本人が構造化・判断を返し、AIが下調べ・記録・分析を担う。非同期メッセージ中心
- 単価月数千円×数人。週4時間で3〜5人が上限。信頼がないと申し込まれないのが弱点

### 型③: 会員制の場を運営する(読書会・練習会・コミュニティ)
- 月額500〜5,000円×少人数。本人は場のルールと仕組みの設計者(裏方)
- **初期集客が最大の壁**。基盤ゼロからいきなりは不可

### 型④(除外): SaaS開発 — フルコミット必須のため週4時間では非現実的
### 型⑤(進化先): メソッドのAI化 — 自分のフレームワークをAIに学習させ拡張(2〜3ヶ月目以降)

## 3. 横断的な成功パターン

1. **分業の鉄則**: AI=定型・下書き・分析・記録 / 本人=独自性(体験・データ)・編集・最終判断・関係構築
2. **基盤ゼロの初手はプラットフォーム内検索**(note/Kindle/ココナラ等)。SNSフォロワー育成は年単位
3. **ビルドインパブリック**: 実践記録の公開自体が、コンテンツ兼集客兼信頼構築(匿名でも成立)
4. **初速は小さい**: 検証可能な事例は初月数千円から。数ヶ月継続で万単位へ
5. **当事者性が最強の差別化**: 成功者は全員、自分の痛みや実践を事業化している
6. **成功者は型①→②→③の順に段階接続している**(コンテンツで認知→1対1→場)

## 4. トレンド調査の要点(ニーズ増大領域)

| 領域 | 根拠 | ニッチ機会 |
|---|---|---|
| 組織行動・管理職 | 課長の94.9%がプレイングマネージャー(産能大)、管理職希望16.7%のみ(パーソル総研)、法人研修は飽和だが個人向けは空白 | **管理職の匿名壁打ち(匿名性が価値そのもの)** |
| 持久系スポーツ | ラン人口758万人、男性40-50代のみ増加(笹川スポーツ財団)。トライアスロン競技人口ピークは40-50代 | データドリブン×非同期コーチング、時間がない大人ランナー特化 |
| 家計・家庭運営 | 共働き1,300万世帯、夫婦の8割が「お金の話が負担」。2027年こども支援NISAで制度移行期 | 夫婦のお金対話設計、出産準備の意思決定伴走 |
| 知識管理・教養 | Obsidian商用無償化(2025)で拡大期。読書会開催数2021年比1.8倍 | 非エンジニア向けObsidian導入代行(空白) |
| AI活用支援 | 中小AI導入率12-20%、最大障壁「何から始めるか分からない」 | ペルソナ特化必須(競合急増・価格競争気味) |

### 横断的示唆
AIが処理・分析・情報提供を代替するほど、「言いにくいことを言う」「本音を引き出す」「関係性の中で成立する言葉」の価値が人間側に残る。AIは基本ユーザーを肯定するため、**「それはやめた方がいい」と言える人間の壁打ち相手**の価値が相対的に上昇する。

## 5. 演繹: プロファイルとの掛け算で導いた設計原則

1. 基盤ゼロ×匿名×週末2-4h → 初手は型①(プラットフォーム集客)、ただし**初月から人との関わり(型②/③の芽)を組み込む**
2. 「場の設計×自己実践」→ 最終形は型③、入口は自分の実践記録
3. 独自性3本(PM構造化・管理職当事者性・実践データ)を必ずコンテンツに注入
4. 消耗回避 → 同期対応・感情労働・即応を要求される設計は除外
