# 工程3 マーケティング調査: 投入プロンプトの記録

2026-07-10。マーケティング調査は社長から統括AIに一任されたため(工程3計画・社長承認済み)、プロンプトの事前承認は省略し、本文書で事後検証可能な形で全量を記録する。調査エージェントは案別に sonnet 5体を並列投入した。

## 共通の構成(全5プロンプトに含めた要素)

1. **役割**: マーケティング調査エージェント。対象事業案の需要と競合の実態を WebSearch で調査
2. **売り手の条件を明示**: 匿名(ペンネーム)の個人、週末2〜4時間稼働、初期予算月1万円以下、日本国内向け
3. **規律**: 「この案は売れる」という結論ありきの調査を禁止。**需要の証拠と反証(売れない兆候・競合の強さ・無料代替の充実)の両方**を集めることを義務化
4. **調査項目5点**: ①需要の証拠(悩み・支払いの表出) ②競合と代替手段(直接競合・無料代替・価格帯) ③ニッチの空き ④販路の適性(note/Kindle/ココナラ等・匿名可否・手数料) ⑤検証可能性ラベル(高=公開データ・大手メディア取材/中=一部客観データ/低=自己申告のみ)の全主張への付与と、実在確認不能な合成事例の除外
5. **報告形式**: 各主張に出典名、末尾に使用検索クエリの全一覧(発注側の網羅性検証用)

## 案別の追加指示

| 案 | 追加指示 |
|---|---|
| 案1 複式簿記家計教材 | 複式簿記×家計の書籍・note・教材の充実度、自動連携アプリ等の無料代替、テンプレ販売の実績例を重点確認 |
| 案2 働きまくりの人の健康習慣 | 「多忙な会社員限定」「ウェアラブル実データ」「仕組み調整型(励まし・伴走なし)」の3切り口の飽和度、健康系の表現規制(医療・薬機法)を確認 |
| 案3 家庭防災PMキット | 無料代替(自治体・消防庁・NHK等)の充実を「正直に評価すること」と明示。「PM化・テンプレ」「ハザードマップ読み解き」「乳児家庭特化」の切り口の空きを重点確認 |

| 案5 管理職の実践記録 | 大手研修・ビジネス書の激戦区である可能性を「正直に評価すること」と明示。「共感型が主流か・逆張り系は存在するか」「匿名当事者の実録形式の競合と売れ行き」「勤務先特定リスクへの一般的対処」を重点確認 |
